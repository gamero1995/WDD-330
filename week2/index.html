    The following words are reserved word in JS. This means that you should not use them to name variables, etc.
    abstract, await, boolean, break, byte, case, catch, char, class

    Primitave data types are: String, Symbol, Number, Boolean, Undefined, Null

    If you assign a non-primitive value to a variable, then this is done by reference, so any changes that are subsequently made will affect all references to that object: 
    
c = { value: 1 };
    
let d = c; // c.value = 1, d.value = 1
    
d.value = 2; // c.value = 2, d.value = 2
    You can add Javascript inside your string by placing it inside curly braces with a $ character in front of them. This is then evaluated and the result is returned in the string output.

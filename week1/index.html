    When looping through an array. It will be better if you use a For each. You will not have to specify the size fo the array. It will do a certain action according to how many object the array contains. See Example below.

    Code Examples

    Looping through an array using a For Each

                
var fruits = ["apple", "orange", "cherry"];
fruits.forEach(myFunction);

function myFunction(item, index) {
  document.getElementById("demo").innerHTML += index 
  + ":" + item + "
    "; 
}
